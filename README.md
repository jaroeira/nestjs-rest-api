## Description

This simple REST-API implements the backend of a Article repository. 

Features:

- User signup;
- Authentication (JWT with refresh token);
- Role based authorization (user / Admin);
- TypeORM - ORM (using PostgreSQL);
- Article CRUD including upload of images;
- Unit tests and e2e tests

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
